## 📋 Descripción del curso

Este repositorio contiene los laboratorios prácticos del curso **Minería de Datos (EIN132A25)**, dictado en el 3.° semestre de la carrera en la USM.

El curso proporciona una comprensión integral de los conceptos, técnicas y herramientas para implementar procesos de minería de datos. Al finalizarlo, los estudiantes serán capaces de aplicar metodologías y algoritmos para la extracción de patrones y conocimiento a partir de grandes volúmenes de datos, evaluar y validar modelos, e implementar soluciones en distintos contextos de datos estructurados y no estructurados.

**Prerrequisitos:** Preprocesamiento de Datos (EIN122A25) · Estadística Descriptiva (MAT007A25)

## 📚 Contenidos por bloque

### Bloque 1 — Fundamentos y Modelos Clásicos (Labs 1–10)

| Lab | Tema | Contenidos clave |
|-----|------|-----------------|
| 01 | Setup + Python para datos | Entorno, Jupyter, NumPy, Pandas básico |
| 02 | EDA — Dataset real | Exploración, estadísticas descriptivas, Titanic dataset |
| 03 | Visualización | Matplotlib, Seaborn, distribuciones, correlaciones |
| 04 | Limpieza de datos | Valores faltantes, outliers, encoding, normalización |
| 05 | Clasificación I | Árbol de decisión, Naive Bayes, aprendizaje supervisado |
| 06 | Evaluación de modelos | Métricas: accuracy, precision, recall, F1, matriz de confusión |
| 07 | Comparación de modelos | SVM, validación cruzada, selección de modelo |
| 08 | Clustering — K-Means | Aprendizaje no supervisado, inercia, elbow method |
| 09 | Clustering avanzado | DBSCAN, clustering jerárquico, silhouette score |
| 10 | Reducción de dimensionalidad | PCA, t-SNE, visualización de alta dimensionalidad |

### Bloque 2 — Datos No Estructurados y Técnicas Avanzadas (Labs 11–16)

| Lab | Tema | Contenidos clave |
|-----|------|-----------------|
| 11 | Minería de texto | TF-IDF, tokenización, clasificación de texto, NLP básico |
| 12 | Series temporales | ARIMA, tendencias, estacionalidad, predicción |
| 13 | Datos geográficos | GeoPandas, análisis espacial, visualización de mapas |
| 14 | Proyecto integrador | Pipeline completo: datos → modelo → evaluación → conclusiones |
| 15 | Preparación certamen | Repaso, ejercicios tipo certamen, preguntas frecuentes |
| 16 | Cierre del curso | Reflexión, buenas prácticas, próximos pasos |
